# simple-blog
Using NuxtJS, fastapi and postgresql


install python requirements by 

`cd backend`<br/>
`pip install -r requirements.txt`<br/>
<br/>
run with <br/>
`python3 start.py`
<br/>

*don't forget to make .env in /backend folder with template provided*

install nodejs requirements by
`cd frontend` <br/>
`npm install` <br/>
<br/>
run with <br/>
`npm run dev`
